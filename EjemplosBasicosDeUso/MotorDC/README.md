# Motor DC

## Descripción
Ejemplo básico de uso de un motor de corriente continua (DC) con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para controlar un motor DC.

## Uso
1. Conecta el motor con driver (L298N, ULN2003, etc.)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la rotación del motor

## Sensores/Actuadores
- Motor de corriente continua (DC)

## Librerías necesarias
- Ninguna especial (se puede usar LedControl)
