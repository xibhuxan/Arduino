# Actuador Analógico

## Descripción
Ejemplo básico de uso de un actuador analógico (potenciómetro, sensor, etc.) con Arduino.

## Esquema
Ver archivo: `actuadorAnalogico.fzz` o `actuadorAnalogico.jpg`

## Código
`Main.ino` - Código básico de uso del actuador.

## Uso
1. Conecta el actuador según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el comportamiento en el monitor serial o LED asociado

## Sensores/Actuadores
- Potenciómetro o sensor analógico

## Librerías necesarias
- Ninguna especial
