# Botón Contador Fácil

## Descripción
Ejemplo básico de uso de un botón con lógica simplificada para contador.

## Esquema
Ver archivo: `botonContadorFacil.fzz` o imagen asociada

## Código
`Main.ino` - Código básico para contador de pulsos del botón.

## Uso
1. Conecta el botón según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Pulsa el botón para incrementar el contador

## Sensores/Actuadores
- Botón con lógica de contador simplificado

## Librerías necesarias
- Ninguna especial
