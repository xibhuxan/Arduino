# Potenciómetro PWM

## Descripción
Ejemplo básico de uso de un potenciómetro para controlar velocidad PWM con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para controlar PWM con potenciómetro.

## Uso
1. Conecta el potenciómetro y el LED/actuador PWM
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Gira el potenciómetro para variar la velocidad

## Sensores/Actuadores
- Potenciómetro
- LED o actuador PWM

## Librerías necesarias
- Ninguna especial
