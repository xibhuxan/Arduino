# Ultrasonido

## Descripción
Ejemplo básico de uso de sensor ultrasónico (HC-SR04) con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para medir distancia con sensor ultrasónico.

## Uso
1. Conecta el sensor ultrasónico HC-SR04
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las mediciones de distancia en el monitor serial

## Sensores/Actuadores
- Sensor ultrasónico HC-SR04

## Librerías necesarias
- Ninguna especial (o NewPing para versión optimizada)
