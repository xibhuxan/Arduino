# Botón con Resistencia

## Descripción
Ejemplo básico de uso de un botón con resistencia externa para evitar el pull-up interno.

## Esquema
Ver archivo: `botonResistencia.fzz` o `botonResistencia.jpg`

## Código
`Main.ino` - Código básico para leer estado del botón con resistencia.

## Uso
1. Conecta el botón según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Pulsa el botón y observa la lectura en el monitor serial

## Sensores/Actuadores
- Botón físico con resistencia externa

## Librerías necesarias
- Ninguna especial
