# IRmovimiento (Detección de Movimiento IR)

## Descripción
Ejemplo básico de uso de un sensor de movimiento infrarrojo con Arduino.

## Esquema
Ver archivo: `IRmovimiento.fzz` o `IRmovimiento.jpg`

## Código
`Main.ino` - Código básico para detectar movimiento con sensor IR.

## Uso
1. Conecta el sensor IR según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Acércate al sensor y observa la detección en el monitor serial

## Sensores/Actuadores
- Sensor de movimiento IR (KY-039 o similar)

## Librerías necesarias
- Ninguna especial
