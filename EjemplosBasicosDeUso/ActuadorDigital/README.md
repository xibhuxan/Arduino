# Actuador Digital

## Descripción
Ejemplo básico de uso de un actuador digital (LED, rele, etc.) con Arduino.

## Esquema
Ver archivo: `actuadorDigital.fzz` o `actuadorDigital.jpg`

## Código
`Main.ino` - Código básico de uso del actuador digital.

## Uso
1. Conecta el actuador al pin indicado en el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el encendido/apagado del actuador

## Sensores/Actuadores
- LED, Relé o otro actuador digital

## Librerías necesarias
- Ninguna especial
