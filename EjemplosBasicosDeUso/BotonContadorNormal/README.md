# Botón Contador Normal

## Descripción
Ejemplo básico de uso de un botón con implementación estándar de contador con debounce.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para contador de pulsos con debounce.

## Uso
1. Conecta el botón según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Pulsa el botón para incrementar el contador

## Sensores/Actuadores
- Botón con debounce estándar

## Librerías necesarias
- Ninguna especial (se puede usar XibDebounceEdge)
