# Encoder

## Descripción
Ejemplo básico de uso de un encoder rotativo con Arduino.

## Esquema
Ver archivo: `encoder.fzz` o `encoder.jpg`

## Código
`Main.ino` - Código básico para leer pulsos del encoder.

## Uso
1. Conecta el encoder según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Gira el encoder y observa los pulsos en el monitor serial

## Sensores/Actuadores
- Encoder rotativo (KY-040 o similar)

## Librerías necesarias
- Ninguna especial
