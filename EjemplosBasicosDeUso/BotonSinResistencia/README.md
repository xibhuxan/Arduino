# Botón sin Resistencia

## Descripción
Ejemplo básico de uso de un botón utilizando la resistencia pull-up interna de Arduino.

## Esquema
Ver archivo: `botonPullup.fzz` o `botonPullup.jpg`

## Código
`Main.ino` - Código básico para leer estado del botón con pull-up interno.

## Uso
1. Conecta el botón sin resistencia externa (pin a GND a través del botón)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Pulsa el botón y observa la lectura en el monitor serial

## Sensores/Actuadores
- Botón físico con resistencia pull-up interna

## Librerías necesarias
- Ninguna especial
