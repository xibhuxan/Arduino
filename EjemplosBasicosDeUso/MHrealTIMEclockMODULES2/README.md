# Reloj MH-real-time clock (DS1307)

## Descripción
Ejemplo de uso del módulo de reloj real time (RTC) DS1307 con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para leer y escribir hora del reloj RTC.

## Uso
1. Conecta el módulo RTC I2C
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la hora en el monitor serial

## Sensores/Actuadores
- Módulo RTC DS1307 o similar

## Librerías necesarias
- Adafruit_SSD1306 (para display)
- MyRealTimeClock o DS1307
