# Botonera

## Descripción
Ejemplo básico de uso de una botonera (múltiples botones) con Arduino.

## Esquema
Ver archivo: `botonera.fzz` o `botonera.jpg`

## Código
`Main.ino` - Código básico para leer estado de la botonera.

## Uso
1. Conecta los botones según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Pulsa los botones y observa las lecturas en monitor serial

## Sensores/Actuadores
- Múltiples botones

## Librerías necesarias
- Ninguna especial (se puede usar XibDebounceEdge)
