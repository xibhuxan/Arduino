# Sensor Analógico

## Descripción
Ejemplo básico de uso de un sensor analógico (LED, temperatura, luz, etc.) con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para leer sensor analógico.

## Uso
1. Conecta el sensor analógico
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las lecturas en el monitor serial

## Sensores/Actuadores
- Varios tipos de sensores analógicos

## Librerías necesarias
- Ninguna especial (o Adafruit_Sensor según el caso)
