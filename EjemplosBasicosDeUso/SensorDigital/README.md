# Sensor Digital

## Descripción
Ejemplo básico de uso de un sensor digital (botón, interruptor, etc.) con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para leer sensor digital.

## Uso
1. Conecta el sensor digital
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las lecturas en el monitor serial

## Sensores/Actuadores
- Sensores digitales (botones, interruptores, etc.)

- Ninguna especial
