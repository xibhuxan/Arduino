# LED Interno Mejorado

## Descripción
Ejemplo básico de uso de los LEDs internos de Arduino con control de tiempo mejorado.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para controlar LEDs internos con temporizador.

## Uso
1. Abre `Main.ino` en tu editor Arduino
2. Compila y carga el código
3. Observa el comportamiento de los LEDs internos

## Sensores/Actuadores
- LEDs integrados de Arduino

## Librerías necesarias
- Ninguna especial (se puede usar XibTimer)
