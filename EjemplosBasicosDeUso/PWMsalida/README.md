# PWMsalida

## Descripción
Ejemplo básico de uso de salida PWM (modulación por ancho de pulso) con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para generar señales PWM.

## Uso
1. Abre `Main.ino` en tu editor Arduino
2. Compila y carga el código
3. Observa el control PWM en los pines configurados

## Sensores/Actuadores
- Salidas PWM de Arduino

## Librerías necesarias
- Ninguna especial
