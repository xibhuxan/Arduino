# LED Externo

## Descripción
Ejemplo básico de uso de un LED externo (no incluido en la placa Arduino) con Arduino.

## Esquema
Ver archivo: `ledExterno.fzz` o `ledExterno.jpg`

## Código
`Main.ino` - Código básico para controlar un LED externo.

## Uso
1. Conecta el LED externo con resistor limitador
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el encendido/apagado del LED externo

## Sensores/Actuadores
- LED externo

## Librerías necesarias
- Ninguna especial
