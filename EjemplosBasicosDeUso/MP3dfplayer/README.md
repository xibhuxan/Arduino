# MP3dfplayer

## Descripción
Ejemplo básico de uso de un módulo DFPlayer Mini para reproducir MP3 con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para reproducir archivos MP3.

## Uso
1. Conecta el módulo DFPlayer Mini con control de botones
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Inserta tarjeta SD con archivos MP3 y reproduce

## Sensores/Actuadores
- DFPlayer Mini (reproductor MP3)

## Librerías necesarias
- DFRobotDFPlayerMini
