# IRmando (Control Remoto Infrarrojo)

## Descripción
Ejemplo básico de uso de un control remoto infrarrojo con Arduino.

## Esquema
Ver archivo: `IRmando.fzz` o `IRmando.jpg`

## Código
`Main.ino` - Código básico para leer señales del control remoto.

## Uso
1. Conecta el receptor IR según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Prueba con el control remoto y observa la decodificación en monitor serial

## Sensores/Actuadores
- Receptor infrarrojo (KY-022 o similar)

## Librerías necesarias
- IRremote
