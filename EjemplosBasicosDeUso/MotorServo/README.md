# Motor Servo

## Descripción
Ejemplo básico de uso de un servo motor con Arduino.

## Esquema
Ver archivo asociado en la carpeta

## Código
`Main.ino` - Código básico para controlar un servo motor.

## Uso
1. Conecta el servo motor al pin PWM
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el movimiento del servo

## Sensores/Actuadores
- Servo motor

## Librerías necesarias
- Servo
