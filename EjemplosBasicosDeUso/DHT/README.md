# Sensores DHT11/DHT22

## Descripción
Ejemplo básico de uso del sensor de temperatura y humedad DHT11/DHT22 con Arduino.

## Esquema
Ver archivo: `dht.fzz` o `dht.jpg`

## Código
`Main.ino` - Código básico para leer temperatura y humedad con sensor DHT.

## Uso
1. Conecta el sensor DHT al pin digital indicado (con pull-up resistor)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las lecturas de temperatura y humedad en el monitor serial

## Sensores/Actuadores
- DHT11 o DHT22 (Temperatura y Humedad)

## Librerías necesarias
- DHTlib (en carpeta Librerias/)
