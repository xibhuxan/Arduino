# Motor paso a paso 3

## Descripción
Ejemplo básico de uso de un motor paso a paso con Arduino.

## Código
`Main.ino` - Código básico para controlar un motor paso a paso.

## Uso
1. Conecta el motor paso a paso con driver (ULN2003, A4988, etc.)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la rotación del motor paso a paso

## Sensores/Actuadores
- Motor paso a paso (28BYJ-48 o similar)

## Librerías necesarias
- LedControl (para drivers ULN2003)
