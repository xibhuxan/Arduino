# NTC

## Descripción
Ejemplo de lectura de sensor NTC (termistor) para medición de temperatura.

## Código
`Main.ino` - Código para medir temperatura con NTC.

## Uso
1. Conecta el sensor NTC
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las lecturas de temperatura en el monitor serial

## Sensores/Actuadores
- Sensor NTC (termistor)

## Librerías necesarias
- Ninguna especial
