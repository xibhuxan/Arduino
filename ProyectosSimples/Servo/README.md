# Servo

## Descripción
Ejemplo de control de servo motor con Arduino.

## Código
`Main.ino` - Código para controlar servo motor.

## Uso
1. Conecta el servo motor
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el movimiento del servo

## Sensores/Actuadores
- Servo motor
