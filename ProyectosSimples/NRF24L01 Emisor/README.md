# NRF24L01 Emisor

## Descripción
Ejemplo de emisor NRF24L01 para comunicación inalámbrica con Arduino.

## Código
`Main.ino` - Código básico para el emisor NRF24L01.

## Uso
1. Conecta un módulo NRF24L01 como emisor
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Envía datos al receptor NRF24L01

## Sensores/Actuadores
- Módulo NRF24L01 (modo emisor)

## Librerías necesarias
- Ninguna especial
