# MotorPWM

## Descripción
Ejemplo de control de velocidad de motor usando PWM con Arduino.

## Código
`Main.ino` - Código para controlar motor con PWM.

## Uso
1. Conecta el motor con driver y driver PWM
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Controla la velocidad del motor con PWM

## Sensores/Actuadores
- Motor DC con control PWM

## Librerías necesarias
- Ninguna especial
