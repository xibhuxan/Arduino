# LCD 16x2 I2C dos pantallas

## Descripción
Ejemplo de uso de pantalla LCD 16x2 I2C con dos líneas de texto.

## Código
`Main.ino` - Código básico para usar LCD I2C.

## Uso
1. Conecta el LCD 16x2 I2C al Arduino
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las dos pantallas del LCD

## Sensores/Actuadores
- LCD 16x2 I2C (dos pantallas)

## Librerías necesarias
- LiquidCrystal_I2C
