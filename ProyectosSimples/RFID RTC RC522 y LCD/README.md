# RFID RTC RC522 y LCD

## Descripción
Ejemplo de sistema de identificación RFID RC522 con reloj RTC y visualización LCD.

## Código
`Main.ino` - Código para leer RFID y mostrar en LCD.

## Uso
1. Conecta el módulo RFID RC522, RTC y LCD
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Acerca una tarjeta RFID y observa la lectura

## Sensores/Actuadores
- RFID RC522
- RTC DS1307 o similar
- LCD 16x2 I2C

## Librerías necesarias
- MFRC522
- MyRealTimeClock o similar
- LiquidCrystal_I2C
