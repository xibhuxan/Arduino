# Inclinómetro

## Descripción
Ejemplo de medición de inclinación con acelerómetro y visualización en LCD.

## Código
`Main.ino` - Código para medir inclinación.

## Uso
1. Conecta el sensor de inclinación y el LCD
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Inclina el dispositivo y observa las lecturas en LCD

## Sensores/Actuadores
- Sensor de inclinación (accelerómetro)
- LCD 16x2

## Librerías necesarias
- Ninguna especial
