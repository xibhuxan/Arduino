# ShieldEthernetV3

## Descripción
Ejemplo de uso de shield Ethernet V3 para conexión a internet con Arduino.

## Código
`Main.ino` - Código básico para conexión Ethernet V3.

## Uso
1. Conecta el shield Ethernet V3
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la conexión al servidor

## Sensores/Actuadores
- Shield Ethernet V3

## Librerías necesarias
- Ethernet
