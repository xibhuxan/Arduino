# DHT11 y LCD

## Descripción
Ejemplo de lectura de temperatura y humedad con sensor DHT11 y visualización en LCD 16x2.

## Código
`Main.ino` - Código para leer DHT11 y mostrar en LCD.

## Uso
1. Conecta el sensor DHT11 y el LCD I2C
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las lecturas en el LCD

## Sensores/Actuadores
- DHT11 (Temperatura y Humedad)
- LCD 16x2 I2C

## Librerías necesarias
- Adafruit_Sensor
- Adafruit_DHT (o similar)
- LiquidCrystal_I2C
