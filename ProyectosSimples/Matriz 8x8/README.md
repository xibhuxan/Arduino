# Matriz 8x8

## Descripción
Ejemplo de control de matriz LED 8x8 con Arduino.

## Código
`Matriz.ino` - Código para controlar matriz 8x8.

## Uso
1. Conecta la matriz LED según el esquema
2. Abre `Matriz.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el comportamiento de la matriz

## Sensores/Actuadores
- Matriz LED 8x8

## Librerías necesarias
- LedControl
