# Contador RPM

## Descripción
Ejemplo básico de medición de revoluciones por minuto (RPM) con Arduino.

## Código
`ContadorRPM.ino` - Código específico para medir RPM.

## Uso
1. Conecta el sensor para medir RPM
2. Abre `ContadorRPM.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa las mediciones de RPM en el monitor serial

## Sensores/Actuadores
- Sensor de RPM

## Librerías necesarias
- Ninguna especial
