# Maqueta

## Descripción
Ejemplo de control de maqueta con Arduino (sensores, motores, etc.)

## Código
`Main.ino` - Código básico para la maqueta.

## Uso
1. Conecta los sensores y motores según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Prueba el control de la maqueta

## Sensores/Actuadores
- Sensores de detección
- Motores

## Librerías necesarias
- Ninguna especial
