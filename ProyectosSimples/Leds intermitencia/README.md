# Leds intermitencia

## Descripción
Ejemplo de control intermitente de LEDs con Arduino.

## Código
`Main.ino` - Código para crear parpadeo en LEDs.

## Uso
1. Abre `Main.ino` en tu editor Arduino
2. Compila y carga el código
3. Observa el parpadeo de los LEDs

## Sensores/Actuadores
- LEDs integrados o externos

## Librerías necesarias
- Ninguna especial
