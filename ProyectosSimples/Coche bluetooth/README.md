# Coche bluetooth

## Descripción
Ejemplo básico de control de un coche mediante comunicación Bluetooth con Arduino.

## Código
`Main.ino` - Código básico para controlar el coche bluetooth.

## Uso
1. Conecta el coche con módulos Bluetooth HC-05/HC-06
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Controla el coche desde el móvil u otro dispositivo bluetooth

## Sensores/Actuadores
- Módulo Bluetooth
- Motor DC (ruedas del coche)

## Librerías necesarias
- Ninguna especial
