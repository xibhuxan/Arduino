# Llama

## Descripción
Ejemplo de simulación de llama con LED o display.

## Código
`Main.ino` - Código básico para simular llama.

## Uso
1. Conecta el display o LED según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el efecto de llama

## Sensores/Actuadores
- LED o display para efecto visual

## Librerías necesarias
- LedControl o Adafruit_GFX
