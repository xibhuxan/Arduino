# Relé

## Descripción
Ejemplo de control de relé con Arduino para conmutar cargas eléctricas.

## Código
`Main.ino` - Código para controlar relé.

## Uso
1. Conecta el relé con driver (optoacoplador, transistor)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Controla la carga conectada al relé

## Sensores/Actuadores
- Relé

## Librerías necesarias
- Ninguna especial
