# Mando

## Descripción
Ejemplo de control remoto o mando con Arduino.

## Código
`Main.ino` - Código básico para el mando.

## Uso
1. Conecta el receptor o transmisor según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Prueba el control del mando

## Sensores/Actuadores
- Módulo de control remoto

## Librerías necesarias
- Ninguna especial
