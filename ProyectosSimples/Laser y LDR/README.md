# Laser y LDR

## Descripción
Ejemplo de uso de láser y sensor LDR (Light Dependent Resistor) con Arduino.

## Código
`Main.ino` - Código para detectar láser con LDR.

## Uso
1. Conecta el módulo láser y el sensor LDR
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Dirige el láser al sensor y observa la detección

## Sensores/Actuadores
- Láser (KY-008)
- LDR (fotorresistencia KY-018)

## Librerías necesarias
- Ninguna especial
