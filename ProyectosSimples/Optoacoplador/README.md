# Optoacoplador

## Descripción
Ejemplo de uso de optoacoplador para aislamiento eléctrico con Arduino.

## Código
`Main.ino` - Código para controlar optoacoplador.

## Uso
1. Conecta el optoacoplador según el esquema
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la señal aislada

## Sensores/Actuadores
- Optoacoplador

## Librerías necesarias
- Ninguna especial
