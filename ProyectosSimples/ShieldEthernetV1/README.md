# ShieldEthernetV1

## Descripción
Ejemplo de uso de shield Ethernet V1 para conexión a internet con Arduino.

## Código
`Main.ino` - Código básico para conexión Ethernet.

## Uso
1. Conecta el shield Ethernet V1
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la conexión al servidor

## Sensores/Actuadores
- Shield Ethernet V1

## Librerías necesarias
- Ethernet
