# Coche simple

## Descripción
Ejemplo básico de control de un coche con Arduino usando pines digitales.

## Código
`Main.ino` - Código básico para controlar el coche simple.

## Uso
1. Conecta el coche con motores y drivers
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Controla el coche desde el puerto serial

## Sensores/Actuadores
- Motor DC (ruedas del coche)

## Librerías necesarias
- Ninguna especial
