# Encoder y LCD

## Descripción
Ejemplo de lectura de encoder rotativo con visualización en LCD 16x2.

## Código
`Main.ino` - Código para leer encoder y mostrar en LCD.

## Uso
1. Conecta el encoder y el LCD I2C
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Gira el encoder y observa la lectura en LCD

## Sensores/Actuadores
- Encoder rotativo
- LCD 16x2 I2C

## Librerías necesarias
- Ninguna especial (o LiquidCrystal_I2C)
