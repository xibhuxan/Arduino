# LCD 16x2 I2C simple

## Descripción
Ejemplo básico de uso de pantalla LCD 16x2 I2C con una sola línea de texto.

## Código
`Main.ino` - Código básico para usar LCD I2C.

## Uso
1. Conecta el LCD 16x2 I2C al Arduino
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el LCD en funcionamiento

## Sensores/Actuadores
- LCD 16x2 I2C (una sola línea)

## Librerías necesarias
- LiquidCrystal_I2C
