# MH-real-time clock ds1302n

## Descripción
Ejemplo de uso del módulo reloj real time (RTC) DS1302 con Arduino.

## Código
`Main.ino` - Código básico para leer y escribir hora del reloj RTC.

## Uso
1. Conecta el módulo RTC DS1302
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la hora en el monitor serial

## Sensores/Actuadores
- Módulo RTC DS1302

## Librerías necesarias
- MyRealTimeClock o similar
