# LCD 16x2 I2C una pantalla

## Descripción
Ejemplo de uso de una sola pantalla LCD 16x2 I2C con Arduino.

## Código
`Main.ino` - Código básico para usar LCD I2C.

## Uso
1. Conecta el LCD 16x2 I2C al Arduino
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la pantalla LCD en funcionamiento

## Sensores/Actuadores
- LCD 16x2 I2C (una pantalla)

## Librerías necesarias
- LiquidCrystal_I2C
