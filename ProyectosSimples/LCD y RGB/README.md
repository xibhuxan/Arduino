# LCD y RGB

## Descripción
Ejemplo de uso de pantalla LCD 16x2 con control de LED RGB.

## Código
`Main.ino` - Código básico para usar LCD y controlar RGB.

## Uso
1. Conecta el LCD y el LED RGB
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el LCD y los cambios de color del LED RGB

## Sensores/Actuadores
- LCD 16x2 I2C
- LED RGB

## Librerías necesarias
- LiquidCrystal_I2C
- LedControl
