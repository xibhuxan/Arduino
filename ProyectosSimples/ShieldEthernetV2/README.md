# ShieldEthernetV2

## Descripción
Ejemplo de uso de shield Ethernet V2 para conexión a internet con Arduino.

## Código
`Main.ino` - Código básico para conexión Ethernet V2.

## Uso
1. Conecta el shield Ethernet V2
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa la conexión al servidor

## Sensores/Actuadores
- Shield Ethernet V2

## Librerías necesarias
- Ethernet
