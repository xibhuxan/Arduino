# PCA9685PW

## Descripción
Ejemplo de uso de controlador PCA9685 para PWM de 16 canales con Arduino.

## Código
`Main.ino` - Código para controlar PCA9685.

## Uso
1. Conecta el módulo PCA9685
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Controla múltiples servos o LEDs PWM

## Sensores/Actuadores
- PCA9685 (controlador PWM 16 canales)

## Librerías necesarias
- Ninguna especial (o similar para control PWM)
