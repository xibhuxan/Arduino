# Matriz 8x8 MAX7219

## Descripción
Ejemplo de control de matriz LED 8x8 usando módulo MAX7219 con Arduino.

## Código
`Main.ino` - Código para controlar matriz MAX7219.

## Uso
1. Conecta el módulo MAX7219 con la matriz LED
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Observa el comportamiento de la matriz

## Sensores/Actuadores
- Matriz LED 8x8 con MAX7219

## Librerías necesarias
- Ninguna especial (o tm1638 para control de display multiplexor)
