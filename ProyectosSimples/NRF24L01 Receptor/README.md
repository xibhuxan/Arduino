# NRF24L01 Receptor

## Descripción
Ejemplo de receptor NRF24L01 para comunicación inalámbrica con Arduino.

## Código
`Main.ino` - Código básico para el receptor NRF24L01.

## Uso
1. Conecta un módulo NRF24L01 como receptor
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Recibe datos del emisor NRF24L01

## Sensores/Actuadores
- Módulo NRF24L01 (modo receptor)

## Librerías necesarias
- Ninguna especial
