# LDR

## Descripción
Ejemplo de uso de sensor LDR (fotorresistencia) con Arduino.

## Código
`Main.ino` - Código para medir luminosidad con LDR.

## Uso
1. Conecta el sensor LDR
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código
4. Cambia la luz sobre el sensor y observa las lecturas

## Sensores/Actuadores
- LDR (fotorresistencia)

## Librerías necesarias
- Ninguna especial
