# NRF24L01 Chat

## Descripción
Ejemplo de comunicación inalámbrica NRF24L01 para chat entre dos Arduinos.

## Código
`Main.ino` - Código básico para chat inalámbrico.

## Uso
1. Conecta dos módulos NRF24L01 (uno como emisor y otro como receptor)
2. Abre `Main.ino` en tu editor Arduino
3. Compila y carga el código en ambos
4. Introduce mensajes y observa la comunicación

## Sensores/Actuadores
- Módulos NRF24L01

## Librerías necesarias
- Ninguna especial
