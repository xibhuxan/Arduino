## 📚 Mis Librerías Personalizadas

Las librerías creadas por mí se caracterizan por tener `Xib` en el nombre:

| Librería | Funcionalidad |
|----------|---------------|
| 🎵 **XibDebounceEdge** | Evita el debounce en botones digitales |
| 🎼 **XibStave** | Interpreta pentagramas musicales en Arduino |
| ⏱️ **XibTimer** | Ejecuta funciones cada X tiempo |
| 📊 **XibTM1638** | Control fácil de placas TM1638 |
| 🖥️ **XibXcreen** | Gestor de botones gráficos Adafruit LCD |
